# My Personal Site

Nothing fancy. Just works.

### Dependencies

- [crunchwrap](https://github.com/egladman/crunchwrap)
- [bashmon](https://github.com/egladman/bashmon)

### Build

Auto-update
```
./run.sh
```

Manual
```
./build.sh
```

### Deploy

```
./deploy.sh
```

### Other

convert profile.jpg -quality 50 -resize 1000x1000 profile_compressed.jpg
